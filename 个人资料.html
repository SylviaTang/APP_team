个人资料

1. 所有个人资料编辑都此页面编辑


2. 头像、个人主页背景支持相机或相册


3. 个性签名14字符，多出不能输入，支持颜文字符号，Emoji表情


4. 用户类别：分为普通用户和认证用户。认证用户需要通过申请认证才能显示。




修改昵称

1. 点击昵称，进入修改昵称页面


2. 昵称长度限定：不得超过15字符，多出不能输入，不支持颜文字符号和Emoji表情


3. 点击取消，返回至个人资料页，昵称未变


4. 点击完成，进入个人资料页，昵称更新





更改头像

1. 点击头像，进入头像设定中页面


2. 头像设定支持相机拍摄或选择图片


3. 如果用户未开启相机权限，则第一次按下相机将会出现弹窗










点击“好”，进入摄像页面；点击“不允许”，返回相册页面


3. 点击取消，返回至个人资料，头像未变


4. 点击图片，进入图片选取页。用户可以放大缩小图片；点击取消，返回相册页面；点击选取，头像更换成功，进入个人资料页

更改性别

1. 点击性别，页面下方将弹出窗口


2. 用户可滚动选择性别选项


3. 选中性别后，点击完成，弹窗消失，个人资料页面中”性别“更新


4. 若在性别设定中页面点击取消，则返回个人资料页，性别未更改





性别设定中

修改昵称

1. 点击个性签名，页面下方将弹出手机键盘，同时整个页面将稍微向上移动


2. 个性签名长度限定：不得超过15字符，多出不能输入，支持颜文字符号和Emoji表情


3. 文字编辑完成，点击输入法中的“Done”即完成个性签名更新






更改生日日期

1. 点击生日，页面下方将弹出窗口


2. 用户可分别滚动年，月，日，选择日期


3. 选择好日期后，点击完成，弹窗消失，个人资料页面中”生日“更新


4. 若在生日日期设定中页面点击取消，则返回个人资料页，生日未更改





生日日期设定中

修改职业

1. 点击职业，进入选择职业页面


2. 用户可以根据自身情况选择职业，被选中项会有         显示在选项最后


3. 点击取消，返回至个人资料页，职业未变


4. 点击完成，进入个人资料页，职业更新





更改地区

1. 点击地区，页面下方将弹出窗口


2. 用户可分别滚动省份和城市


3. 选择省份和城市后，点击完成，弹窗消失，个人资料页面中”地区“更新


4. 若在地区设定中页面点击取消，则返回个人资料页，地区未更改
